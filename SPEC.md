# SPEC — starttmux

> Status: **mini MVP** (5 practice levels)

## Current scope

- Desktop keyboard simulator with tmux prefix (Ctrl+b)
- Mobile touch mode: tap keycaps in order (auto on coarse pointer, or `?mode=mobile`)
- 5 unlocked practice levels — no daily lock, no progression gate
- Commands: `%` `"` arrows `c` `n` `p` `0`–`9`
- Win screen: time + mistakes; separate `localStorage` for desktop vs mobile bests
- Dark/light theme

## Not in this release

Hints, share block, payments, daily scheduler, streaks, accounts, server-side game API.

## Possible next steps

Daily puzzle mode, more levels, hint tiers, spoiler-free share, optional tip CTA.
