# SPEC — starttmux

> Status: **mini MVP** (5 practice levels + cheat sheet)

## Current scope

- Desktop keyboard simulator with tmux prefix (Ctrl+b)
- Mobile touch mode: tap keycaps in order (auto on coarse pointer, or `?mode=mobile`)
- **Golden path** for AI coders — 5 steps teaching agent + shell + logs layout (display order differs from level IDs)
- **Cheat sheet** at `cheatsheet.html` — shortcuts, path, recommended layout, appendix for real tmux
- Commands in game: `%` `"` arrows `c` `n` `p` `0`–`9`
- Win screen: time + mistakes; next-step link along golden path
- Separate `localStorage` for desktop vs mobile bests
- Dark/light theme

## Not in this release

Hints, share block, payments, daily scheduler, streaks, accounts, server-side game API.

## Possible next steps
